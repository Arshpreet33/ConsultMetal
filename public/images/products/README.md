# Product Images

This folder contains images for product pages.

## Required Images

### Product Hero
- `products-hero.jpg` - Hero background image for products listing page and detail pages

### Product Cards
- `nesting-software.jpg` - Main image for Nesting Software product
- `cad-cam-software.jpg` - Main image for CAD/CAM Software product

### Fallback
- `placeholder.jpg` - Placeholder image used when product images are not available

## Image Specifications

- **Hero Images**: Recommended size 1920x500px
- **Product Images**: Recommended size 800x480px (5:3 aspect ratio)
- **Format**: JPG or PNG
- **Optimization**: Compress images for web to improve loading times

## Notes

Please add your product images to this folder matching the filenames above. The application will automatically use these images for the respective products.
