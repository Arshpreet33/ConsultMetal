# Services Images

This directory contains images for the Services pages.

## Required Images

### Hero Image
- **services-hero.jpg** - Background image for services listing and detail pages hero section
  - Recommended size: 1920x1080px (16:9)
  - Format: JPG or PNG
  - Content: Manufacturing/industrial related imagery

### Service Images
Each service requires an image with 5:3 aspect ratio (e.g., 800x480px):

1. **workflow-design.jpg** - Workflow Design service
2. **product-analysis.jpg** - Product Analysis service
3. **software-recommendation.jpg** - Software Recommendation service
4. **software-development.jpg** - Software Development service
5. **capital-expenditure.jpg** - Capital Expenditure Recommendation service
6. **cad-development.jpg** - CAD Development service

### Placeholder
- **placeholder.jpg** - Fallback image shown when service image fails to load
  - Size: 800x480px (5:3 aspect ratio)
  - Content: Generic placeholder or "Image coming soon"

## Image Guidelines

- **Format**: JPG (for photos) or PNG (for graphics with transparency)
- **Aspect Ratio**: 5:3 for service cards
- **File Size**: Optimize to under 200KB per image for fast loading
- **Naming**: Use lowercase with hyphens, matching the service slug from config
- **Content**: Professional, high-quality images representing each service
